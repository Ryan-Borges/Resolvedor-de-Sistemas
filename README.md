# Resolvedor de Sistemas Lineares e Matrizes em Python

Este projeto é um programa desenvolvido em Python que permite resolver sistemas de equações lineares e calcular determinantes de matrizes. A interação ocorre por meio do terminal, com entrada de dados feita pelo usuário.

## Objetivo
Projeto criado com fins educacionais para praticar:
- Lógica de programação
- Manipulação de matrizes
- Aplicação de conceitos de álgebra linear
- Organização do código em funções
- Entrada de dados via terminal

## Tecnologias utilizadas
- Python
- NumPy

## Como executar
1. Certifique-se de ter o Python instalado
2. Instale a biblioteca NumPy (`pip install numpy`)
3. Execute o arquivo principal `.py` pelo terminal

## Estrutura do Projeto
1. Arquivo principal `.py` → contém o menu e as funcionalidades do programa
2. Arquivo `.ipynb` → versão do projeto desenvolvida no Google Colab para testes e demonstração
